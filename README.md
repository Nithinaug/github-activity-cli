# GitHub Activity CLI (Go)

A simple command-line tool written in **Go** that displays basic GitHub activity for a user, such as the number of push events and repositories pushed to.

This project is intentionally minimal and is meant to demonstrate how **CLI applications work in Golang** using positional arguments (no flags, no external libraries).

---

## Features

- Fetch public GitHub activity for a user
- Display:
  - Total push events
  - Unique repositories pushed to
- Simple argument-based CLI
- Uses GitHub public API
- Beginner-friendly and lightweight

---

## Tech Stack

- **Language:** Go
- **API:** GitHub Public REST API
- **CLI Input:** `os.Args`

---

## Installation

### Prerequisites
- Go 1.18 or later
- Internet connection
